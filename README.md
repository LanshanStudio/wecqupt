## 分支管理

```
wecqupt
  ├─ master       // 默认分支（**开发测试版本所用分支**；保护分支，只允许由[other]分支-Pull Requests-reviews而来，故无法push）
  ├─ stable       // 稳定分支（**正式版本所用分支**；高级保护分支）
  └─ [other]      // 其他开发分支（只允许该项目Collaborators创建及push分支）
```
